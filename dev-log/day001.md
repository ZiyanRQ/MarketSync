## Challenge 1 - Harvesting Too Quickly

### Problem

When I first automated harvesting on a 1×1 grass farm, the farmer harvested faster than the grass could regrow.

### Solution

```python

```

inside the loop to slow the farmer down.

### Lesson Learned

## Challenge 3 - Growing Two Crops Together

### Problem

Once my farm became larger, growing only one crop became inefficient because by the time I reached the start again everything had already regrown.

I wanted to maximise the available farming space by growing two crops together.

Initially I ran into several bugs:

- Grass was growing instead of bushes.
- Trees were planting on the wrong tiles.
- My planting pattern wasn't consistent.

### Solution

I learned about using the modulo operator (`%`) together with the farmer's X and Y position.

This allowed me to create a repeating planting pattern.

```python
if get_pos_x() % 2 == 1 and get_pos_y() % 2 == 0:
    plant(Entities.Tree)
elif get_pos_x() % 2 == 0 and get_pos_y() % 2 == 1:
    plant(Entities.Tree)
else:
    plant(Entities.Bush)
```

### Lesson Learned

Breaking a problem into logical conditions makes debugging much easier.
